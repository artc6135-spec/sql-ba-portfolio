# SQL & Business Data Analysis Portfolio

This repository contains SQL production scripts, relational database models, and analytical queries designed for Business Analyst (BA) and Data Analyst (DA) domains.

## Repository Structure

* /automation-risk-analysis — Analytical scripts identifying AI impact on the global labor market by 2030 (PostgreSQL).
* /retail-analytics — Inventory optimization and supply chain data processing scripts.
* /legacy-refactoring — Optimization logs turning high-latency queries into scalable production code.

## Technical Competencies

* Dialects: PostgreSQL / MS SQL Server.
* Analytical Tools: Window functions (ROW_NUMBER, DENSE_RANK, LEAD/LAG), CTE (Common Table Expressions), complex multi-stage JOIN operations.
* Optimization: Execution plan analysis, subquery elimination, and explicit data type conversion.

## Current Project: AI Market Impact Analysis
* Objective: Evaluate salary distributions against automation probabilities.
* Target Metrics: Automation rank within risk categories, high-salary risk exposure.
