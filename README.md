# SQL & Business Data Analysis Portfolio

This repository contains production-ready SQL scripts, database models, and architectural insights demonstrating advanced data manipulation capabilities for Business Analyst (BA) and Data Analyst (DA) domains. 

The core philosophy governing this portfolio is **"Artifacts over activity"** — delivering optimized, verifiable, and documented code to solve real-world business bottlenecks.

## 🚀 Repository Structure

* 📂 `/banking-analytics` — Financial datasets processing, customer segmentation (RFM), and transaction data quality scripts (optimized for core banking operations).
* 📂 `/retail-foodtech` — Supply chain analysis, logistics optimization, and performance tracking queries.
* 📂 `/refactored-legacy` — Production-level optimization of legacy academic queries, transitioning from implicit layouts to explicit relational operations.

## 🛠 Technical Stack & SQL Capabilities

* **Dialects:** PostgreSQL / MS SQL Server.
* **Relational Operations:** Advanced Multi-stage `JOIN` logic, explicit relational scoping.
* **Data Aggregation & CTEs:** Complex Common Table Expressions for modular, readable query architecture.
* **Analytical Windows:** Comprehensive use of window functions (`ROW_NUMBER()`, `DENSE_RANK()`, `LEAD/LAG`, `PARTITION BY`) for trend analysis and financial slicing.
* **Optimization:** Query execution plan analysis, redundant subquery elimination, and explicit data type casting.
